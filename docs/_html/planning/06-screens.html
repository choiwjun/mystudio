상태 표시 (D2 — 정상 상태는 중립색):
  - Good (초록): 게시 준비 가능 글 있음 — 긍정 신호
  - Warning (주황): 검수 실패 있음 — 경보 신호
  - Risk (빨강): 🚨 즉시 대응 필요 (컴플라이언스 위반 등)
  - Focus (중립 회색 #6B7280): 아직 주제 선택 안 함 — 진행 중
  - Revenue (파랑): 오늘 수익 기록 필요 — 정보

우측: 버튼 2~3개 (색상 #2563EB 또는 #1F2937) + 미기록 배지 (D6)
  - 미기록 배지: 게시 후 성과 미기록 콘텐츠 수 (예: "📊 3개 미기록")
  - 배경: #FEE2E2, 텍스트: #991B1B
