09-personas.md: 사용자 페르소나

Paperclip Company OS v0.7 MVP 기준: 1개 페르소나 (사용자 = 오너) 기준일: 2026-07-02

페르소나 1: "일 3시간 블로거가 회사처럼 돌리려는 개인"

기본 정보

이름: 김수민 (가명)
나이: 28세
직업: 블로그 + 부업 프리랜서
거주: 서울

시간 제약:
- 본업: 월~금 9시~5시 (회사 또는 원격)
- 블로그 전담 시간: 주 3일, 하루 1~2시간
- Paperclip 사용 시간: 매일 아침 30분 (제약 스트릭트)

수익 목표: 월 ₩500,000 (용돈 + 저축 목표)
문제: "부업이지만 사업처럼 돌려야 하는데, 혼자는 불가능"

핵심 페인포인트

Pain 1: 시간 부족

아침 10분: 오늘 뭘 쓸지 고민
  → 트렌드? 지난번 잘 된 주제? 뭐가 수익이 될까?
  → 보통 15분 → 30분 지남

오전 30분: 블로그 본문 작성
  → "제목, 첫 화면, 비교표, 링크 배치까지 한 번에"
  → 보통 혼자 하면 1~2시간 소요

오후 15분: 쇼핑커넥트 링크 찾기
  → "아, 이 상품 가격 확인해야지"
  → 링크 3~5개 추가 10분

저녁 20분: SNS 올리기
  → 블로그 복사 + Instagram 포스팅 + Threads
  → 보통 30분 이상

밤 10분: "오늘 수익 얼마였지?"
  → 대시보드 확인, 엑셀 정리

→ 총 2시간 소요 (목표: 30분)

Pain 2: 지속성 부족

현재 상황:
- 월 2~3글 작성 (꾸준하지 못함)
- "주말에 한 번에 써야지" → 안 씀
- "다음 달엔 더 많이" → 실행 안 됨

원인:
1. 매일 아침 "뭘 쓸까" 고민하는 피로
2. 쓸 줄 알지만 "구조"를 맞추는 시간 (제목, 단락, 링크)
3. 성과 기록이 번거로워서 건너뜀
4. 잘 된 패턴을 모르면 다음엔 또 처음부터 시작

→ "혼자" vs "회사처럼" = 지속 불가능

Pain 3: 최적화 불가능

현재 상황:
- "이 제목은 잘 먹혔다" vs "이건 안 먹혔다"를 기억하기만 함
- 다음 글에 반영하려고 해도 체계 없음
- 어떤 상품군이 수익이 되는지 모르기도 함

원인:
- 성과 분석 도구가 없음
- 블로그 분석은 되지만, "Paperclip이 정말 맞게 설계했나?"는 모름
- 고민할 시간이 없음 (쓰느라 바쁨)

결과:
- 매달 반복되는 실패
- 100 조회 정도에서 정체

페르소나 1의 한 줄 요구 (문제 해결의 핵심)

문제 상황

"매일 아침 30분, 오늘 만들 수익형 콘텐츠 1개를 골라서 패키지로 받고 싶다."

더 정확히

언제: 매일 아침 (9시~10시, 출근 전)
뭘: 지난번 잘 된 유형의 제목 + 블로그 구조 + 쇼핑 링크까지
  (처음부터 만들지 말고, 지난 성공 패턴 기반)
왜: 남는 시간에 게시만 하고, 나머지는 자동화가 해줬으면 좋겠다
결과: 월 4개 이상 지속 가능하고, 수익 ₩500k 달성

현재 vs 원하는:
현재:
  아침 10분 고민 → 30분 본문 작성 → 15분 링크 → 20분 SNS → 비효율
  
원하는:
  아침 5분 기회 3개 보고 1개 선택 → 25분 최종 정리 → Export → 게시 수동만

현재: 혼자 하는 블로거 = 모든 게 일
원하는: 혼자지만 회사처럼 = 나는 최종 오너만

페르소나 1의 하루 운영 여정

아침 (9시~9시 30분)

상황: 출근 전 카페에서 스마트폰으로

1. Paperclip HQ 접속
   "오늘 기회 3개" 보여줌
   - 제목, Why Now, 4축 점수
   - 각각 1분 내로 훑음

2. 직감으로 선택
   "이 주제가 좋겠다" → [선택]

3. 1분 대기 (Paperclip이 처리 중)
   그 사이 메일 확인

4. 콘텐츠 패키지 도착
   - HomeFeed 제목 3개 (강한 후킹)
   - Search 제목 1개
   - 구조: 공감 → 비교표 → 상품별 설명 → FAQ
   - 쇼핑 링크 3~5개 (가격도 자동 크롤링)
   - Compliance 통과 ✓

5. 아무것도 직접 작성하지 않음
   "다 되어있네?" (정말 이 느낌이 중요)

→ 출근 (9시 30분 정각)

저녁 (19시~19시 30분)

상황: 집에 와서 랩탑으로

1. Paperclip 콘텐츠 페이지 열기
   - [Export] 클릭 → Markdown 다운로드
   - 또는 [블로그에 복사] → 자동으로 클립보드

2. 네이버 블로그 접속
   - "새 글" → 마크다운 붙여넣기 (30초)
   - 썸네일 이미지 1개 업로드 (1분)
   - "발행" (10초)

3. 성과 기록
   - 게시 URL 복사 → Paperclip에 붙여넣기
   - 조회수 / 클릭 수 (2시간 후 다시 방문해서 입력)

→ 저녁 작업 끝 (총 5분, 이미지 작업 제외)

다음날 아침

상황: 출근 전

1. Paperclip 열기
   - 어제 성과: "조회 250, 클릭 12"
   - 분석: "이 제목 유형은 이번 달 평균 270"
   - 추천: "오늘도 같은 각도로 갈까? 다르게 갈까?"

2. Company Memory 조회
   - 잘 먹힌 패턴 3개
   - 지난번 실패 원인

3. 오늘의 기회 3개 (Hermes가 학습함)
   - 어제 성공 유형과 비슷한 주제 우선 제시
   - 수익 난 카테고리 먼저 보여줌

→ 또 다시 30분 사이클 (이번엔 더 효율적)

페르소나 1의 성공 지표

주간

- 콘텐츠: 1개 이상 작성 (목표: 2개)
- 게시: 100% (작성한 것 다 게시)
- 성과 기록: 100% (매일 저녁)
- Paperclip 사용 시간: 아침 30분 + 저녁 15분

월간

- 콘텐츠: 4개 이상 (목표: 8개)
- 총 조회수: 1,000 이상
- 수익: ₩300,000 이상 (목표: ₩500,000)
- 성공 패턴 저장: 3개 이상

분기 (Month 3)

- 콘텐츠: 12개 (월 4개 기준)
- 평균 조회수: 300/글 (초기 250에서 +20%)
- 총 수익: ₩1,200,000 (월 ₩400k 평균)
- 다음 전략 개선: "검색형 키워드 강화", "제습기 카테고리 집중" 등

페르소나 1의 기능 우선순위

"가장 중요한 건 뭔가?"

Tier 1: 반드시 필요 (없으면 안 씀)

1. HQ 메인 (기회 3개 보기)
   → "매일 아침 이것만 하면 돼"
   
2. Hermes Opportunity Memo (지금 바로 쓸 수 있는 주제)
   → "고민 없이 선택하기"
   
3. Export 기능 (복사 또는 다운로드)
   → "어차피 블로그에 직접 붙여넣어야 함"
   
4. 성과 기록 (숫자 3~4개만 입력)
   → "이걸 안 하면 다음 주제 선택이 맞춱"

Tier 2: 매우 유용 (있으면 좋음)

5. Company Memory (패턴 자동 저장)
   → "다음 주제에 반영되면 좋겠지"
   
6. Compliance Gate (자동 검수)
   → "대가성 문구 놓치지 말아야 하니까"
   
7. 콘텐츠 상세 (전체 뷰, 수정)
   → "혹시 뭔가 이상하면 수정하고 싶은데"

Tier 3: 있으면 더 좋음 (Phase 2)

8. 클립 대본 생성
   → "블로그는 꼭 올려야 하지만, 클립은 여력 있을 때"
   
9. SNS 변환
   → "팔로워 늘리고 싶으면"

페르소나 1의 체험 맵 (Empathy Map)

생각 (Thinks)

"혼자는 한계가 있다"
"좋은 주제를 놓치고 있을까?"
"이게 정말 최적의 구조일까?"
"이번 달 수익이 나올까?"
"매일 지속할 수 있을까?"

느낌 (Feels)

불안감: "혼자가 아닐까?"
피로: "매번 처음부터 시작하는 기분"
궁금증: "왜 저번엔 조회수가 많고 이번엔 적을까?"
희망: "시스템이 있으면 4배는 더 할 수 있을 텐데"
믿음: "AI가 나를 도와줄 수 있을까?"

들음 (Hears)

주변 블로거들:
  "나도 SNS, 블로그 다 하는데 너무 힘들어"
  "수익은 쌀쌀해서, 시간 투자가 안 맞아"
  
소셜미디어:
  "AI가 도와준다더라"
  "자동화하면 월 1000만도 가능"
  
커뮤니티:
  "콘텐츠는 쓰는데 마케팅이 어렵다"

행동 (Behaviors)

아침: Paperclip 접속 (예정)
  → 기회 보고 선택 (자동화되어야 함)

낮: 업무 중 (블로그 터치 X)

저녁: 수동 게시 + 성과 기록 (자동화 불가)

밤: "다음에는..." 고민 (Company Memory가 대신해줄 것)

고통 (Pains)

1. 시간: 혼자 하면 2시간 필요
2. 선택: 뭘 쓸지 매번 고민
3. 지속: 한 번 건너뛰면 3주 건너뜀
4. 최적화: 내가 하는 게 맞는지 모름
5. 수익: 아직도 월 200만 원대?

이득 (Gains)

1. 효율: 아침 30분만 투자
2. 신뢰: "시스템이 골라줘?"
3. 패턴: "다음엔 뭘 해야 할지 알겠네"
4. 수익: 월 500만 원 가능할 것 같은데?
5. 지속: "이렇게 하면 한 달, 3개월, 6개월 갈 수 있을 것 같다"
