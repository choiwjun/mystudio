## 1. HQ 메인 (Paperclip HQ Command Center)

### 목적

매일 아침 접속했을 때 **"오늘 뭘 할 거야?"를 5초 안에 판단**하고, 기회를 선택해서 제작까지 30분 안에 완료하도록 만드는 화면.

### 전체 레이아웃

```
┌──────────────────────────────────────────────────────────────────────────┐
│ Paperclip OS  [2026-07-02 (수)]  Status: Good  │  [오늘 브리핑 생성] [설정] │
├──────────────┬──────────────────────────────────────┬────────────────────┤
│              │                                      │                    │
│ 사이드바     │    중앙 메인 (4개 블록)             │   우측 패널         │
│              │                                      │   (4개 섹션)       │
│ • HQ         │ 1) 오늘의 경영 브리핑                │                    │
│ • Hermes     │    └─ [오늘 주제 선택하기]           │ 1) Owner           │

│ • Compliance │    (카드 리스트 3~5개)               │ 2) Compliance      │

│              │                                      │                    │

```

### 1.1 헤더

```
┌─────────────────────────────────────────────────────────────────┐
│ 🔗 Paperclip Company OS    2026.07.02 (수)    Status: ⚫ Good   │
│                                                                  │
│ 우측: [📋 오늘 브리핑 생성]  [⚙️ 설정]  [👤 로그아웃]           │
└─────────────────────────────────────────────────────────────────┘

헤더 높이: 60px
좌측: 로고 + 제목 (고정, 스크롤 안 함)
중앙: 오늘 날짜 (자동 갱신)

상태 표시 (D2 — 정상 상태는 중립색):
  - Good (초록): 게시 준비 가능 글 있음 — 긍정 신호
  - Warning (주황): 검수 실패 있음 — 경보 신호
  - Risk (빨강): 🚨 즉시 대응 필요 (컴플라이언스 위반 등)
  - Focus (중립 회색 #6B7280): 아직 주제 선택 안 함 — 진행 중
  - Revenue (파랑): 오늘 수익 기록 필요 — 정보

우측: 버튼 2~3개 (색상 #2563EB 또는 #1F2937) + 미기록 배지 (D6)
  - 미기록 배지: 게시 후 성과 미기록 콘텐츠 수 (예: "📊 3개 미기록")
  - 배경: #FEE2E2, 텍스트: #991B1B
```
